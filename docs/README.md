# Orbiton Documentation Site

This is the official documentation and API reference site for the Orbiton Server Management Panel, built using React + Vite and designed for maximum performance, responsiveness, and premium visual aesthetics.

## 🚀 Quick Deploy to Vercel

You can deploy this documentation site directly on **Vercel** with a single click, or manually import this directory.

### Configuration on Vercel Dashboard

When setting up your Vercel project manually, configure the following settings:

1. **Framework Preset:** `Vite` (Vercel will detect it automatically).
2. **Root Directory:** `docs` (Make sure to point to the `docs` folder inside this repository).
3. **Build Command:** `npm run build`
4. **Output Directory:** `dist`

### 💻 Local Development

To run this documentation site locally:

```bash
cd docs
npm install
npm run dev
```

## 🎨 Design Systems
* **Dark Neon Cyber theme** for zero eye-strain and futuristic aesthetics.
* **Responsive 3-column layout** featuring Category-scoped Navigation, central documentation canvas, and an interactive **On this page** dynamic Table of Contents.
* **Zero external markdown dependencies:** Utilizes an integrated, ultra-lightweight custom JavaScript Markdown renderer supporting inline styling, fencing, headers, and code block copy control.
